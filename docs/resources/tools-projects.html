כלים ופרויקטים חישוביים

כלים בסיסיים שישמשו אותנו לתרגול עקרונות חישוביים

  1. Plain text (editors): visual studio code; notepad++; sublime text

  2. Command line interface & Git: git for windows (https://www.atlassian.com/git; https://ohmygit.org/press/) 

  3. Bibliography: Bibtex; CSL; Zotero, online libraries: DH, computational literary studies, doing DH, digital archaeology, digital history

  4. OCR: Tesseract engine tools; OCR.net; Transkribus

  5. Data cleaning: OpenRefine

  6. Text analysis and annotation: voyant tools (based on Spyral); Recogito (LOD); CATMA (general use); LEAF-VRE (TEI/XML); topic modeling tool (based on Mallet); jsLDA: In-browser topic modeling; AntConc; Dicta (NLP)

  7. Data visualization (general): Tableau; Flourish; RAWGraphs; Palladio 

    1. Mapping tools: QGIS; Allmaps; StoryMap; open history map 

    2. Network analysis: Network Navigator; Gephi; Gephi Lite; Cytoscape

    3. Advanced applications: D3 and P5 java script libraries; text visualization browser; mapshaper

  8. Coding environment for beginners: Rstudio desktop and cloud (R and python); Jupyter (python); Anaconda (many)

*כלים דיגיטליים לחקר טקסט מפורטים תחת פרויקט TAPoR; להדרכות והדגמות כיצד להשתמש בכלים שנלמד בקורס ואחרים ראו את הבלוג של מרים פוזנר וההיסטוריון המתכנת והאתר של פרויקט מינימלי של לימוד מדעי הרוח הדיגיטליים בכלא(!); רשימה מודרכת של פרויקטים וחומרי הדרכה על פי נושאים במדעי הרוח הדיגיטליים נמצאת באתר ה-MLA
